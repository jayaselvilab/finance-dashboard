📊 Advanced Finance Dashboard
A full-stack MERN application designed to help users track their financial health through a visual and interactive interface.

🚀 Live Demo
Frontend: https://finance-dashboard-nine-swart.vercel.app

Backend API: Hosted on Render

✨ Features
Real-time Dashboard: Summarizes Total Income, Expenses, and Balance.

Visual Analytics: Interactive Pie Chart using Recharts to visualize spending vs. earnings.

Transaction History: A clean table showing all recent financial activities.

Cloud Database: Powered by MongoDB Atlas for 24/7 data availability.

Responsive Logic: Frontend built with React Hooks for seamless state updates without page refreshes.

🛠️ Tech Stack
Frontend: React.js, CSS3, Recharts

Backend: Node.js, Express.js

Database: MongoDB Atlas (NoSQL)

Deployment: Vercel (Frontend), Render (Backend)
